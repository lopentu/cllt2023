給分方式

計分比例如下

項目 百分比
課堂參與與練習 10%
作業 40% (10% x 4) 選最優十份作業
期末專題 40%
助教實習課 10%

課堂期待

  1. 自主學習與積極參與討論
  2. 確實預習
  3. 不遲交、不遲到
  4. 彼此協助與合作
  5. 誠信與負責

遲交評分

  • 每份作業最遲在截止日後三天補交，除非有正當理由教師認可，否則皆扣 10%。

  • 期末專案不得遲交。
